## Installation / Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/nayem-ahmedz/bite-share.git
   ```
2. Navigate to the project folder:
   ```bash
   cd bite-share
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
